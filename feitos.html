    Instalação - Biblioteca | Script
  • Biblioteca @types: npm install @types/express -D
  • Biblioteca ts-node: npm install ts-node -D - Para executar código typeScript
  • Biblioteca typeScript: npm install typescript -D
  • Biblioteca ts-node-dev: npm install ts-node-dev -D - Detecta alterações no código automáticamente
  • Script de execução: "dev": "ts-node-dev src/server.ts" - Executar o comando npm run dev para executar o APP
